Tecnologias

O que é CSS?

Linguagem de estilização, usada para estilizar os elementos de HTML da página.

O que é VS-code?

Editor de código versátil e amplamente usado por desenvolvedores.

O que é HTML?

Linguagem de marcação de texto, usada para estruturar todos os elementos da página.


Tecnologias usadas

HTML (Hypertext Markup Language): Linguagem de marcação para estruturar o conteúdo de páginas web, usando elementos como tags para definir títulos, parágrafos, links e outros elementos.

CSS (Cascading Style Sheets): Linguagem de estilo que controla a apresentação visual de documentos HTML, permitindo definir cores, fontes, layout e outros estilos para criar um design coeso.

Visual Studio Code: Ambiente de desenvolvimento integrado (IDE) da Microsoft, leve e altamente extensível, utilizado para escrever, editar e depurar código em várias linguagens, especialmente popular para desenvolvimento web.
